This tag is used to indicate a link.

img

Back To Top

This tag is used to indicate an image.

This tag includes the attribute src which stands for source. This links the browser to wherever your image is being stored and allows the computer to retrive and display it.

It also includes the attribute alt which allows you to write text describing the image. This is important incase the browser is unable to laod or image, or in the case someone is using a screen reader--the description can be read to them.

< img src="YourImageSourceLinkHere" alt="A Description of Your Image Here" >
